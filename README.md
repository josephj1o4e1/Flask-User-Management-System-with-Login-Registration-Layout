# Python Flask User Management System
This is a Flask Website that implements a basic user management system with Login/Register layout.   

Framework: Flask  
Session management: Flask-Login  
OAuth: oauthlib  
ORM: SQLAlchemy + Alembic  
Bootstrap 5.3
